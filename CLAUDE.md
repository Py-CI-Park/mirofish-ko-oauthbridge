# CLAUDE.md

이 저장소에서 작업하는 에이전트는 `AGENTS.md`를 우선 참고합니다.

## 커밋 메시지 규칙
- 사용자가 커밋을 요청하면 커밋 메시지는 **한국어 + 마크다운 본문** 형식으로 작성합니다.
- 첫 줄은 짧은 의도 문장으로 작성합니다.
- 본문에는 가능하면 아래 섹션을 포함합니다.
  - `## 배경`
  - `## 변경 내용`
  - `## 검증`
- 저장소 Lore 규칙에 따라 trailer는 영문 키를 유지합니다.
  - `Constraint:`
  - `Rejected:`
  - `Confidence:`
  - `Scope-risk:`
  - `Directive:`
  - `Tested:`
  - `Not-tested:`

## 검증 및 안내
- 커밋 전에는 실제 실행 결과를 확인합니다.
- 서버 설정 변경이 포함되면 최종 답변에 재시작 안내를 포함합니다.
- 검증하지 않은 항목은 `Not-tested:`에 명시합니다.
